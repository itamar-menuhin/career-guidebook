This vault is the source of truth for Career Compass content. Edit notes here, then run `npm run build:vault` to regenerate the site artifacts.

- Flow steps live in **01_Flow/**
- Focus areas and buckets live in **02_Focus-Areas/**
- Recommendation cards live in **03_Cards/**
- Common pathways live in **04_Common-Pathways/**
- Session templates live in **05_Templates/**

Use the `_Templates` folder for Obsidian templates when creating new notes.
